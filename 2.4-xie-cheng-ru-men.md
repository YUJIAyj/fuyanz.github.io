# 2.4协程入门

## 简单计时器

每隔两秒在控制台中显示当前游戏经历的时间。

```csharp
Using UnityEngine;
public class Test:MonoBehaviour{
    void Start(){
        //开启一个协程，协程函数为Timer
        StartCoroutine(Timer());
        }
    //协程函数
    IEnumerator Timer(){
        //不断循环执行，但是并不会导致死循环
        while(true){
            //打印四个汉字
            Debug.Log("测试协程");
            //等待1秒
            yield return new WaitForSeconds(1);
            //打印当前游戏经历的时间
            Debug.Log(Time.time);
            //再等待1秒
            yield return new WaitForSeconds(1);
            }
        }
    }
        
```
